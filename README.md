# House Price Prediction Using Linear Regression

## Project Overview

This project demonstrates the implementation of a simple Linear Regression machine learning model to predict house prices based on the area of the property.

The project was developed using Python and popular data science and machine learning libraries as part of my learning journey in Machine Learning.

## Objective

The objective of this project is to understand the fundamentals of supervised machine learning and Linear Regression by analyzing the relationship between house area and price.

## Dataset

The dataset contains two variables:

* `area` — The area of the house, used as the independent variable.
* `price` — The price of the house, used as the dependent/target variable.

## Model

The project uses the Linear Regression algorithm.

The basic equation of Linear Regression is:

`y = mx + c`

Where:

* `y` = predicted house price
* `x` = house area
* `m` = coefficient/slope
* `c` = intercept
